MeetUp

MeetUp app is a serverless, progressive web application(PWA) built with ReactJS using TDD(Test driven development) approach. The application uses the Google Calendar API to fetch upcoming events and displays a list of upcoming events. Users can filter events by city, specify the number of events to display, show/hide event details, and app offline feature is available.
